1- Télecharger le fichier phpDocumentor.phar ici : https://phpdoc.org/phpDocumentor.phar 
2- Enregistrer ce fichier à la racine du projet 
3- Taper la commande : php phpDocumentor.phar run -d . -t docs/api

==> Un répertoire /docs a du être créé... Il continet un fichier index.html que vous pouvez 
afficher ==> C'est la documentation technique de votre projet
